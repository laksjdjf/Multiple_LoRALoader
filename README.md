# Multiple_LoRALoader
LoRAを複数適用できます。

multiple_lora_loaderにいっぱいノードが追加されます。

normalizeをTrueにして、normalize_sumをxにすると、strengthの合計をxにする、みたいな感じにできます。

（プラスのstrengthとマイナスのstrengthがある場合は利用を避けた方がいいと思います。）

config.txtで適用できるLoRAの数一覧を指定できます。
